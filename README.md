# Linux Syscheck
A lightweight shell script designed to audit disk usage, memory usage and the users logged in the system

## Features 
* **Disk usage auditing:** Displays disk usage for a given path.
* **User tracking:** Checks for the users currently logged into the session.
* **Memory monitoring:**Displays current free and used system memory in human-readable format.

## Prerequisites

Before running this script, ensure your system meets the following requirements:

* **OS:** Any modern Linux distributions (Debian, Ubuntu, CentOS, Fedora, etc.).
* **Shell:** Bash shell environment

## Installation
Clone the repository to your local machine

```bash
git clone git@github.com:Hms1000/linux-syscheck.git
cd linux-syscheck
```

## Usage 
Make the script Executable and run it

```bash
chmod +x syscheck.sh
```

```bash
./linux-syscheck.sh
```
